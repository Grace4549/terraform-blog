## What is Infrastructure as Code (IaC)?

Infrastructure as Code (IaC) is a way to **define and manage cloud infrastructure using code** instead of manual steps.  
Normally, setting up servers, databases, and networks is slow, error-prone, and hard to repeat. IaC solves this by making infrastructure **repeatable, automated, and version-controlled**.

## Declarative vs Imperative Approaches

- **Imperative:** You give step-by-step instructions manually.  
- **Declarative:** You declare the **desired state**, and the tool figures out how to achieve it.  

Terraform is declarative — you tell it what you want, and it creates it automatically.

## Why Terraform is Worth Learning

Terraform works with **AWS, Azure, GCP**, and more, making it an essential DevOps skill.  
It lets you manage infrastructure in **code**, so changes are safe, auditable, and reproducible.
